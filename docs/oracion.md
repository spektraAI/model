## 1. Clasificación por Intención del Hablante

| Tipo de Oración | Función | Ejemplo en Inglés | Traducción |
| :--- | :--- | :--- | :--- |
| **Declarative** | Declarar o negar un hecho. | *I like coffee.* | Me gusta el café. |
| **Interrogative** | Realizar una pregunta. | *Do you like coffee?* | ¿Te gusta el café? |
| **Imperative** | Dar una orden o hacer una petición. | *Drink your coffee.* | Bebe tu café. |
| **Exclamatory** | Expresar una emoción o sorpresa. | *This coffee is great!* | ¡Este café está genial! |

---

## 2. Clasificación por Estructura Sintáctica

| Estructura | Definición | Ejemplo en Inglés |
| :--- | :--- | :--- |
| **Simple** | Una sola cláusula independiente. | *The cat slept.* |
| **Compound** | Dos cláusulas independientes unidas por una conjunción (*and, but, or*). | *The cat slept, and the dog barked.* |
| **Complex** | Una cláusula independiente más una dependiente (*because, although*). | *The cat slept because it was tired.* |
| **Compound-Complex** | Dos cláusulas independientes y al menos una dependiente. | *The cat slept and the dog barked because it was loud.* |

---

## 3. Oraciones Condicionales

| Tipo | Uso | Ejemplo en Inglés |
| :--- | :--- | :--- |
| **Zero Conditional** | Verdades universales o hechos científicos. | *If it rains, the grass gets wet.* |
| **First Conditional** | Situaciones reales o posibles en el futuro. | *If I study, I will pass the test.* |
| **Second Conditional** | Situaciones hipotéticas o imaginarias. | *If I won the lottery, I would travel.* |
| **Third Conditional** | Situaciones irreales del pasado (arrepentimientos). | *If I had known, I would have helped.* |


## 4. Estructuras Avanzadas y de Énfasis

| Estructura | Función | Ejemplo en Inglés |
| :--- | :--- | :--- |
| **Passive Voice** | Resalta el objeto de la acción. | *The project was completed.* |
| **Cleft Sentence** | Añade énfasis a un elemento. | *It was then that I knew.* |
| **Tag Question** | Busca confirmación. | *She is coming, isn't she?* |
| **Wish Sentence** | Expresa deseos o arrepentimientos. | *I wish I were there.* |
| **Reported Speech** | Dice lo que otra persona dijo. | *He said that he was tired.* |
